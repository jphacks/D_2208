# リモコン用 アプリケーション

![version](https://img.shields.io/badge/version-1.0.0__SNAPSHOT-blue.svg)

## 概要

プレゼンテーション用レーザポインターを模したアプリケーションです。

ポインター表示、タイマーの設定、リモートでのスライドの切り替え機能を提供します。

## 開発

### 環境構築

```sh
yarn install
```

### 開発用サーバの起動

```sh
yarn run dev
```

### コードチェック

```sh
yarn run code-check
```

### ビルド方法

```sh
yarn build
```

### 使用技術

- React@^18.2.0
- Vite@^3.1.0
- TypeScript@^4.6.4
