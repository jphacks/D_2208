# ドキュメント

![deploy](https://github.com/membership-console/docs/workflows/deploy/badge.svg)

本リポジトリは Membership Console のドキュメントです。

## ビルド方法

```sh
$ pip install -r requirements.txt
$ mkdocs build --strict
```

## サーバを起動

```sh
$ mkdocs serve
```
